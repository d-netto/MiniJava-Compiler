# MiniJava Compiler

Compiler for a subset of Java introduced by Appel's Modern Compiler Implementation in Java. The BNF (Backus-Naur Form) of this programming language is available in https://www.cambridge.org/resources/052182060X/MCIIJ2e/grammar.htm. 

Files in `test/test_files` were taken from the MiniJava project page (https://www.cambridge.org/resources/052182060X/).
